Getting started

TalkToMeBaby v0.3.2 — install in under a minute, then start listening.

Install from the Marketplace

The easiest way to install is directly from the VS Code Marketplace ↗.

  1. Open the Extensions view in VS Code (Cmd+Shift+X / Ctrl+Shift+X)
  2. Search for TalkToMeBaby
  3. Click Install

Or install from the command line:

code --install-extension rishmadaan.talktomebaby

Install from VSIX

Download the .vsix file from the GitHub Releases page ↗, then install it with the CLI:

code --install-extension talktomebaby-0.3.2.vsix

Or via the UI: Cmd+Shift+P / Ctrl+Shift+P → Extensions: Install from VSIX… → select the file.

Build from source

git clone https://github.com/rishmadaan/talktomebaby.git
cd talktomebaby
npm install
npm run package   # produces talktomebaby-0.3.2.vsix
code --install-extension talktomebaby-0.3.2.vsix

Quick start

  1. Open any .md, .txt, or other supported prose file
  2. Cmd+Shift+P / Ctrl+Shift+P → TalkToMeBaby: Read Document
  3. The Reader panel opens and reading starts from the beginning

Read from here / Read selection

To start from a specific point, place your cursor and run TalkToMeBaby: Read from Here from the right-click menu or command palette. To read a highlighted passage, select text first and run TalkToMeBaby: Read Selection.

Supported file types

.md .mdx .txt .rst .org .tex .adoc

No API key needed to start. On macOS, TalkToMeBaby defaults to the built-in say command — fully offline, official Apple CLI, no data leaves your machine. On Windows and Linux, it defaults to the free Edge TTS provider, which uses an unofficial Microsoft endpoint (no SLA; requires internet). Any provider can be switched at any time from the gear menu in the Reader panel.
