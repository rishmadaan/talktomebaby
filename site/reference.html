Reference

TalkToMeBaby v0.3.2 — authoritative source: package.json contributes.

Commands

All commands are available from the Command Palette (Cmd+Shift+P / Ctrl+Shift+P).

Command ID Title Description
talktomebaby.readDocument TalkToMeBaby: Read Document Open the Reader panel and read the active file from the beginning.
talktomebaby.readSelection TalkToMeBaby: Read Selection Open the Reader panel and read from the start of the current selection.
talktomebaby.readFromCursor TalkToMeBaby: Read from Here Open the Reader panel and read from the cursor position.
talktomebaby.jumpToCursor TalkToMeBaby: Jump Playback to Cursor Jump playback to the word at the cursor in the source editor. Always starts playback, even when paused. Default keybinding: alt+j (editor focus required).
talktomebaby.pauseResume TalkToMeBaby: Pause/Resume Toggle playback between paused and playing. Default keybinding: Cmd+Shift+R / Ctrl+Shift+R.
talktomebaby.stop TalkToMeBaby: Stop Stop playback and close the active reading session.
talktomebaby.openReader TalkToMeBaby: Open Reader Reveal the Reader panel if it is already open (does not start a new read).
talktomebaby.setApiKey TalkToMeBaby: Set API Key Store an API key for ElevenLabs or Sarvam AI. Keys are saved in VS Code SecretStorage (OS credential store) and never written to settings.
talktomebaby.selectProvider TalkToMeBaby: Select TTS Provider Pick a TTS provider from a quick-pick list. The active provider is marked with ✓. Reconfigures the reader in place — no auto-play.
talktomebaby.selectVoice TalkToMeBaby: Select Voice Pick a voice for the active provider from a quick-pick list. Reconfigures the reader in place — no auto-play.

Settings

All settings live under talktomebaby.* in VS Code settings (settings.json or the Settings UI). Most can also be changed live from the Reader panel gear menu.

Key Type Default Description
talktomebaby.provider string "auto" TTS provider. Allowed values:
  • "auto" — macOS say on macOS (offline, official); Edge TTS elsewhere (free, unofficial endpoint).
  • "edge" — Edge TTS: unofficial Microsoft endpoint, free, no key, word-level timing. Requires internet.
  • "elevenlabs" — ElevenLabs: premium neural voices, word-level timing. Requires an ElevenLabs API key.
  • "say" — macOS say: fully offline, official Apple CLI, estimated timing. macOS only.
  • "sarvam" — Sarvam AI: Indian English focus, estimated timing. Requires a Sarvam AI API key.
talktomebaby.voice.edge string "en-US-AriaNeural" Voice ID for Edge TTS. Use TalkToMeBaby: Select Voice to browse all available voices.
talktomebaby.voice.elevenlabs string "21m00Tcm4TlvDq8ikWAM" Voice ID for ElevenLabs. Use TalkToMeBaby: Select Voice to browse voices from your account.
talktomebaby.voice.say string "Samantha" Voice for macOS say. Novelty voices (Zarvox, Boing, etc.) are filtered from the picker. macOS only.
talktomebaby.voice.sarvam string "shubh" Voice for Sarvam AI. Use TalkToMeBaby: Select Voice to browse available voices.
talktomebaby.speed number 1.0 Playback speed multiplier (0.5–2.0). Pitch is preserved at all speeds. Updated automatically when changed in the player bar; persisted across sessions.
talktomebaby.editorClickToJump string "alt-j" How to jump playback from the source editor. Allowed values:
  • "off" — editor jumping disabled.
  • "alt-j" — jump only via the alt+j keyboard shortcut (cursor must be in the editor).
  • "plain-click" — any plain mouse click in the editor during an active session triggers a jump.
Jumping always starts playback from the target word, even when paused.
talktomebaby.readerFontSize number 16 Reader panel font size in px. Adjustable from the Reader gear panel (12–28 px stepper).
talktomebaby.highlight.sentenceColor string "" Sentence band highlight color. Empty string uses the theme default. Set from the Reader gear panel's Appearance section.
talktomebaby.highlight.wordColor string "" Current word highlight color. Empty string uses the theme default. Set from the Reader gear panel's Appearance section.
talktomebaby.cacheSizeMB number 200 Disk cache size limit in MB. Synthesized audio is stored under globalStorageUri/audio-cache with LRU eviction. Increase for large libraries; decrease to conserve disk space.

Menus

TalkToMeBaby adds entries to two VS Code menus. Both are gated to supported file types (.md, .mdx, .txt, .rst, .org, .tex, .adoc).

Editor title bar (editor/title)

A button appears in the editor title bar (the icon row at the top-right of a file tab) when the active file is a supported type.

Command Icon Group When (plain English) When clause
talktomebaby.readDocument $(unmute) (speaker icon) navigation The active file's extension is one of: .md, .mdx, .txt, .rst, .org, .tex, .adoc resourceExtname =~ /\.(md|mdx|txt|rst|org|tex|adoc)$/

Editor right-click context menu (editor/context)

Two entries appear in the right-click context menu when the active file is a supported type.

Command Group When (plain English) When clause
talktomebaby.readFromCursor navigation The active file's extension is a supported type resourceExtname =~ /\.(md|mdx|txt|rst|org|tex|adoc)$/
talktomebaby.readSelection navigation The active file is a supported type and the editor has a non-empty selection editorHasSelection && resourceExtname =~ /\.(md|mdx|txt|rst|org|tex|adoc)$/

Note: talktomebaby.readDocument is also the editor title button; talktomebaby.readSelection only appears in the context menu when text is selected. talktomebaby.readFromCursor is context-menu only — it has no title-bar button.
