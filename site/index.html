Core features

🎯

Dual-layer highlighting

The Reader panel shows a sentence band behind the current sentence and a moving word sweep on top — so you always know exactly where you are.

🖱️

Click any word to jump

Click any word in the Reader panel to jump playback there instantly. Press alt+j from the source editor to jump to your cursor position.

⚡

Speed 0.5–2x, pitch preserved

Change speed on the fly with presets or the fine slider. No re-synthesis — speed is applied locally, so pitch stays natural and it's instant.

💾

200 MB disk cache

Synthesized audio is cached to disk (default 200 MB, LRU eviction). Re-read the same document as many times as you like — no extra API credits spent.

Voices & providers

TalkToMeBaby works out of the box on every platform — no API key required to start.

  • macOS say — the default on macOS. Fully offline; no data leaves your machine. Uses the official macOS speech synthesis command.
  • Edge TTS — the default on Windows and Linux. Free, no key needed. Uses an unofficial Microsoft endpoint — choose this with that in mind.
  • ElevenLabs — high-quality neural voices. Requires an ElevenLabs API key.
  • Sarvam AI — Indian-language voices. Requires a Sarvam AI API key.
